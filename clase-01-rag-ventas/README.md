# Clase 01 — Agente RAG con Mistral

## Descripción
Bot que responde preguntas sobre datos de ventas usando RAG + LangChain + Mistral.

## Archivos
- `01_normalizacion.ipynb` → limpieza y normalización del dataset
- `02_agente_rag.ipynb` → agente RAG con Mistral Small

## Stack
- Python 3.12
- LangChain
- Mistral Small (mistral-small-latest)
- ChromaDB
- Google Colab
